# mutations.txt
# Format: CHAIN,RES_NUM,ORIG_AA,MUT_AA  — one mutation per entry
# For multi-mutant variants, put multiple entries on ONE line separated by +
# Lines starting with # are ignored
#
# Examples:
# Single mutant:
C,31,H,R
# Multi-mutant variant (all applied together to one PDB):
C,31,H,R+C,36,E,D+C,41,E,A
# Another single:
C,36,E,K
